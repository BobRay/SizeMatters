Ems Graph

This chart shows the distribution of visitors by the width of their viewport expressed in Ems. This is my favorite measure because it takes into account the font size being used. It's calculated with this formula: Math.round( width / parseFloat(font_size)); where both width and font_size are expressed in CSS Pixels.

To make the graph readable, values greater than 100 Ems are rounded down to 100.
