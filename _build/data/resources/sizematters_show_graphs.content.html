Who's Visiting with What?

If your web site has a responsive design, you may be interested in fine-tuning the design based on the actual devices your visitors are using. It's generally a bad idea to design for specific devices, but it's nice to know how many of your users users are visiting using phone, tablet, laptop, and desktop devices. The values reflect the viewport width, not the device width, so if the user has the browser smaller than full-screen, the recorded width values will be smaller.

SizeMatters uses JavaScript to ask the browser the viewport width in Ems and CSS pixels, and the current font size in px. The taller the bar in the graph, the more users are visiting with that value. There's more information in the SizeMatters documentation at Bob's Guides.

Take the specific values in the graphs with a grain of salt. There are plenty of browser and device anomalies and results may depend on whether or not you have a meta viewport tag (and what's in it). Still, in my experiments, the values are very close and can almost always be used to identify specific devices (not that you should do that).

The graphs are done with pChart 2.0.
